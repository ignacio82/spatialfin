Smart Playback

SpatialFin is starting to remember how you actually watch.

Smart language preferences and series-level memory now sit alongside XR playback instead of feeling like generic media-player toggles.

Smarter Language Handling

  • Ordered spoken languages: Add and prioritize the languages you understand instead of choosing only one.
  • Original audio preference: SpatialFin can favor likely-original audio for multilingual content.
  • Automatic subtitle fallback: If the selected audio is not one you speak, SpatialFin enables a matching subtitle track when possible.
  • Searchable XR setup: Language configuration uses a larger XR-first dialog with search, ordering, and clearer result visibility.

Series Memory

  • Audio overrides persist: If you switch a series from dub to original audio, future episodes can inherit that choice.
  • Subtitle-track memory: SpatialFin now stores the exact subtitle variant when you correct a bad subtitle pick.
  • Manual corrections compound: The goal is fewer repeated fixes as you move episode to episode.

AI Runtime Reality

  • On-device path included: SpatialFin supports an AICore / Gemini Nano integration path where the runtime is exposed.
  • Galaxy XR fallback today: Current Galaxy XR firmware testing indicates third-party AICore is unavailable, so deterministic voice handling and optional cloud Gemini fallback are the practical path.
  • Diagnostics built in: Voice settings surface AICore package status, permissions, and telemetry summaries for debugging.
  • More detail available: A dedicated page explains the smart playback model in one place.

Everything in the stack supports the experience

Immersive XR

  • Interactive Agency: Grab and move the entire application or video player window anywhere in 3D space.
  • Native XR Controls: Material 3 for XR orbiters float secondary controls in space.
  • XR-First Layouts: Larger typography and labeled action buttons instead of dense icon-only controls.
  • Large-Target Playback UI: Larger controls and a dedicated chapter picker for easier hand-first interaction.

Advanced Playback & Media

  • Stereoscopic 3D: Automatic detection and rendering of SBS, top-bottom, and other 3D formats.
  • Spatial Audio: High-fidelity positional audio that pins sound to the screen's location.
  • Pixel-Perfect Anime Subtitles: Integrated libass JNI renderer for ASS/SSA rendering.
  • Version Selection: Choose between different versions, such as 3D vs. 2D or 4K vs. 1080p.
  • Dynamic Quality: Control streaming bitrate on the fly from 480 Kbps to 120 Mbps.
  • Global Bitrate Settings: Set a default maximum streaming bitrate globally.
  • Smart Audio And Subtitles: Spoken-language ordering, subtitle fallback, and per-series memory reduce repetitive track management.

Voice Control & AI

  • Unified Smart Assistant: Use the hold-to-talk middle-finger pinch gesture to issue playback commands or ask natural language questions about the movie.
  • AI Metadata Chat: Uses on-device AICore where available and falls back to cloud Gemini when the user provides an API key.
  • Contextual Subtitle Awareness: The assistant tracks recent dialogue so you can ask for clarifications on what characters just said.
  • Spatial TTS Feedback: The AI responds aloud with automatic volume ducking, keeping you immersed without reading text.
  • In-Player AI Search: Voice search opens as a spatial overlay to search and launch media seamlessly.
  • Voice Diagnostics: Dedicated settings showing enablement, permissions, AICore package status, and runtime availability.
  • Cloud Key Option: Advanced users can store their own Gemini API key in settings to enable cloud-backed fallback features on devices without AICore.

SyncPlay & Ecosystem

  • SyncPlay Watch Parties: Create, join, refresh, and mirror playback state with Jellyfin SyncPlay groups natively.
  • Jellyfin Integration: Full Jellyfin server connectivity for movies, shows, and collections.
  • Seerr Integration: Connect Jellyseerr or Overseerr to search and request media directly from the app search screen.

Offline & Local Library

  • Automatic Offline Mode: Switches offline when the server is unreachable, then returns online when available.
  • Local Library: Browse and play videos stored directly on the XR device with metadata inference.
  • Reliable Resumable Engine: Robust background downloading that survives app restarts and network swaps.
  • Complete Metadata Mirroring: Offline media now includes all posters, backdrops, and scrubber thumbnails (trickplay).
  • Real-Time Foreground Progress: Track download status directly from the system tray.
  • Configurable Downloads: Download original files or smaller transcoded versions with your preferred options.
  • Smart Download Reconciliation: Automatically cleans up stale entries if files are removed manually.
  • In-App Download Management: Delete downloads from the item screen or Downloads tab.
