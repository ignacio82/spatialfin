Feature Breakdown

Everything the Beam Pro experience includes.

SpatialFin Beam shares the core playback engine with the XR build but adapts the interface and codec pipeline for the Beam Pro's hardware.

Layout

  • Landscape-Locked: The app always launches in landscape orientation, matching how AR glasses display content.
  • Sidebar Navigation: A persistent sidebar with labeled icons for Home, Search, Local, Network, Downloads, Settings, Servers, and Users.
  • Poster Carousels: Horizontal scrolling rows of poster cards for suggestions, continue watching, next up, and library sections.

Design

  • Dark Theme: Material 3 dark color scheme matched to the XR experience for visual consistency across devices.
  • Compact Feature Cards: Media detail cards show poster, title, metadata, overview, and action buttons in a tight horizontal layout.
  • Poster Grid: Five-column grid layout for browsing libraries with artwork and titles at a glance.

Codec Support

  • FFmpeg Software Decoding: EAC3 (Dolby Digital Plus), TrueHD, and other advanced audio codecs decoded in software when the hardware can't handle them.
  • PGS Bitmap Subtitles: Image-based PGS subtitles render correctly alongside text-based formats, so no subtitle tracks are hidden.
  • Libass Styled Subtitles: ASS/SSA anime subtitles with full typesetting rendered via the native libass JNI library.

Track Selection

  • Smart Language Handling: Ordered spoken-language preferences, anime detection, and series-level overrides for audio and subtitle selection.
  • All Tracks Visible: Audio and subtitle tracks that the hardware can't natively decode are still available thanks to software fallback.
  • Direct Play: Streams files directly from the server without transcoding when codecs are supported.

Sidebar Voice Search

  • Mic Button: A microphone icon in the sidebar header triggers on-device speech recognition with a single tap.
  • Live Transcript: Partial speech transcripts appear below the sidebar title in real-time as you speak.
  • Instant Search: When recognition finishes, the app navigates to Search and executes the query automatically.

In-Player Voice

  • Playback Commands: Play, pause, seek, skip intro, next episode, and 30+ other voice actions during playback.
  • AI Assistant: On-device Gemini Nano or cloud Gemini processes natural language for context-aware responses about what you're watching.
  • No Keyboard Needed: Between sidebar voice search and in-player commands, the entire experience can be hands-free.

Jellyfin Integration

  • Full Library Browsing: Movies, shows, seasons, episodes, box sets, and collections from your Jellyfin server.
  • Continue Watching & Next Up: Resume where you left off with dedicated carousels on the home screen.
  • Jellyseerr Requests: Search and request new media through Jellyseerr or Overseerr directly from the search screen.

Network & Local Files

  • SMB & NFS: Browse and stream from network shares with automatic mDNS discovery.
  • Local Storage: Play videos stored directly on the device.
  • TMDB Enrichment: Automatic poster and metadata matching for files on network shares.

Downloads

  • Resumable Downloads: Download movies and episodes for offline playback. Downloads survive app restarts.
  • Original or Transcoded: Choose between keeping the original file or a smaller transcoded version.
  • Progress Tracking: Real-time download progress from the system notification tray.

Offline Mode

  • Automatic Fallback: Seamlessly switches to the offline library when the server is unreachable.
  • Complete Mirroring: Downloaded items include posters, backdrops, and metadata.
  • SyncPlay: Create or join Jellyfin SyncPlay groups to watch with friends in sync.

Getting Started

  • Companion QR Setup: Scan a QR code from the SpatialFin Companion to import servers, users, and preferences instantly.
  • Manual Server Entry: Add Jellyfin servers by URL and authenticate with username and password.
  • Multi-Server & Multi-User: Switch between multiple servers and user accounts from the sidebar.

Configuration

  • Language Preferences: Set preferred audio and subtitle languages, spoken language ordering, and per-series overrides.
  • Subtitle Styling: Font size, text color, and background color for subtitles.
  • Voice Assistant: Toggle voice commands, set verbosity, spoiler policy, and configure cloud AI API keys.
