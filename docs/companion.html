Installation

Deploy in seconds with Docker.

The SpatialFin Companion runs as a lightweight container on your home server or NAS.

1. Clone The Repo

Start by downloading the companion app repository to the machine that will run Docker:

git clone https://github.com/ignacio82/SpatialFin-Companion.git

2. Open The Project

Move into the cloned folder before starting the service:

cd SpatialFin-Companion

3. Deploy Container

Use Docker Compose to start the companion app in the background:

docker compose up -d

4. Configure Hub

Navigate to http://your-ip:1982 and add your Jellyfin servers, users (Username/Password), and network shares.

5. Scan & Enjoy

Open SpatialFin on your XR device, go to Settings -> SpatialFin Companion, and scan the QR code displayed on your dashboard.

Technical Details

A secure, local source of truth.

The companion app communicates exclusively over your local network. It generates a unique setup token for every discovery attempt, ensuring that only authorized scans can access your configuration details.
